This is a repository for the projects based on the makemore tutorial created by Andrej Karpathy. There are some differences between my version and the colab notebooks he created as these are created based on my understanding.As much as possible, I am using the tensors / functions rather than the modules to get some hands on practice with torch


What is achieved in makemore Part 1:
- experience working with the tensor library directly instead of the modules
- experimenting with embedding size to see how it affects accuracy + quality of names generated
- setting up the framework for hyperparameter testing + concurrent training using multiprocessing 
- data cleaning + modification (eg noise removal) + visualisation of data
